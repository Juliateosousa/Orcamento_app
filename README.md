# Orcamento_app
-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
This is an application I developed for the company I work for. I’m using Firebase as the database.

When using the web version, the app doesn’t always handle things very well and can crash sometimes. However, when running through Apple Developer or the Windows app, it works much better, aside from a few bugs that still happen.

The previous system we used was very hard to modify, so for me it was important to make this one as simple as possible for the user. Sometimes that leads to more code and a few extra bugs, but for us that trade-off is acceptable.

Feel free to use it however you prefer.
